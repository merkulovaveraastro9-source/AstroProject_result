# Тёмное гало и бар спиральных галактик: существует ли взаимосвязь?
*Автор: Меркулова Вера, 10-2*  
*Начало работы над проектом: декабрь 2025 года*  

Данный проект направлен на исследование взаимосвязи между размерами бара в галактиках и параметрами их тёмного гало. Программа позволяет обрабатывать большие массивы данных, строить графики зависимостей, определять параметры аналитических зависимостей с помощью метода наименьших квадратов.
